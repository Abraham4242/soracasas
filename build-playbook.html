How Sora uses it

Land diligence and feasibility first (titled vs ROP, survey, utilities), then lock the model + tier + region, which fixes the spec, the price, and the timeline. Design and permits run in parallel; the build moves through seven inspected stages with payments released only against verified progress; the buyer sees weekly video and signs off each milestone. The whole thing is one fixed price, with Sora carrying the contingency so the buyer never sees an overrun.
